# Qasr Invoice Generator

A React + TypeScript invoice generator application built with Vite, Tailwind CSS, and integrated with Google Apps Script for data submission.

## Features

- Invoice form with customer details
- **Text-based PDF generation** (searchable, selectable, and computer-friendly)
- Google Apps Script integration for data submission
- Responsive UI with Tailwind CSS
- TypeScript for type safety

## Requirements

- Node.js (v14 or higher)
- npm (comes with Node.js)

## Installation

1. Navigate to the project directory:
   ```bash
   cd Qasr-Invoice-Generator
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

## Running the App

Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Technologies Used

- React 18
- TypeScript
- Vite
- Tailwind CSS
- jsPDF (for native text-based PDF generation)
- Lucide React
